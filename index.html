O que é o HTML

Como o próprio nome já diz "HyperText Markup Language" - Linguagem de Marcação de Hipertexto - o HTML não é considerado uma linguagem de programação, mas sim, uma linguagem de marcação. Isso ocorre porque um documento HTML não possui uma sequência lógica de comandos; ou seja, não é necessário montar uma estrutura conectada com uma ordem específica para que ele "funcione".

Já na linguagem de programação é necessário um algoritmo para executar uma ação específica.

Elementos

Chamamos de elementos todas as tags que são utilizadas para a contrução do HTML, que consiste em duas partes: Cabeça (head) e corpo (body).

Os elementos principais de um HTML são:

  • html: Utilizado sempre para iniciar um documento, declarando que o mesmo é um HTML;
  • head: Utilizado para cabeçarios;
  • body: Utilizado para desenvolver o corpo, parte visual da página.

Comandos Básicos HTML

Temos algumas tags essenciais (mas não obrigatórias) para a construção de um documento.

Segue algumas:

  • Cabeçalhos: Os cabeçalhos são definidos com as tags h1 até h6.
    1. h1: Definido como títulos mais importantes, normalmente utilizado para títulos principais;
    2. h2: Definido como subtítulos importantes que estão abaixo do h1;
    3. h3: Definido como títulos de seções menores;
    4. h4, h5 e h6: Definido como títulos de seções menores e menos importantes do documento.
  • p: Utilizado para paragráfos;
  • a: Utilizado para links, devemos inserir o atributo href para que o destino do link seja especifícado;
  • img: Utilizado para inserir imagens, devemos inserir o atributo src para especificar o caminho ou nome do arquivo da imagem;
  • ol: Utilizado para listas ordenadas;
  • ul: Utilizado para listas não ordenadas;
  • li: Utilizado para inserir os itens da lista, cada item deve começar com essa tag;
  • blockquote: Utilizado geralmente para citações;
  • strong: Utilizado para textos em negrito;
  • i: Utilizado para textos em itálico;
  • small: Utilizado para textos menores;
  • mark: Utilizado para grifar um texto;
  • u: Utilizado para sublinhar palavras ou textos.

Como incluir links externos e internos

Nos referimos como links externos a qualquer link que não esteja dentro do seu documento HTML, cuja intenção é redirecionar o leitor para outra página. Por outro lado, links internos são aqueles cuja intenção é direcionar para outras partes do mesmo documento HTML. Abaixo, explicaremos com mais detalhes.

Links externos

Para inserirmos links externos devemos utilizar a tag "a" e inserir o atributo "href" para assim inserir o destino do link. Devemos também nomear o título do hiperlink, o qual será o texto clicável que aparece a página.

Links internos

É possível também criar links para páginas da sua HTML, criando novos arquivos e nomeando-os. Isso permite separar a página principal das demais que deseja criar.

Na hora de construir a tag para direcionar para sua outra página, em vez de inserir uma URL, você utilizará o nome da página desejada.

Conclusão

Por mais que a linguagem HTML não seja de programação, ela é fundamental para a criação de páginas web. Utilizando uma estrutura de marcação simples e clara, composta por tags e atributos, o HTML permite definir a estrutura e o conteúdo de uma página, desde textos simples até elementos mais complexos como imagens, links e formulários. Através do uso correto das tags, é possível criar páginas acessíveis e bem estruturadas, garantindo uma experiência de usuário consistente e eficiente.
